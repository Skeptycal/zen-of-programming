
## My favorite free resources for new programmers

### For HTML and CSS

* [Learning CSS Through Creating Art](https://dev.to/aspittel/learning-css-through-creating-art-54c0)
* [CSS Diner](https://flukeout.github.io/)
* [Flexbox Froggy](https://flexboxfroggy.com/)
* [CSS Grid Garden](https://cssgridgarden.com/)

### For JavaScript

* [A re-introduction to JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript/A_re-introduction_to_JavaScript)
* [Khan Academy Computer Programming](https://www.khanacademy.org/computing/computer-programming)
* [JS 30](https://javascript30.com/)
* [You Don't Know JS](https://github.com/getify/You-Dont-Know-JS)

### For other Programming Languages

* [Code Academy](https://www.codecademy.com/)
* [Django Girls](https://tutorial.djangogirls.org/en/)
* [Google's Python Class](https://developers.google.com/edu/python/)
* [Learn Python](https://www.learnpython.org/)
* [Try Ruby](https://ruby.github.io/TryRuby/)

### For Computer Science

* [BaseCS](https://medium.com/basecs) - there are also [podcast](https://www.codenewbie.org/basecs) and [video]() versions of some of these articles!
* [Coding Cheat Sheets](https://github.com/aspittel/coding-cheat-sheets)

### For Code Challenges

* [CodeWars](https://www.codewars.com/)
* [HackerRank](https://www.hackerrank.com/)
* [Project Euler](https://projecteuler.net/)

* [Daily Programmer](https://www.reddit.com/r/dailyprogrammer/)
